Codeggs的常用命令

Git

  1. 将本地新建的项目提交到远程仓库
    1. git init初始化本地仓库
    2. git add .将工作目录内容添加至暂存区
    3. git commit -m "first commit"将暂存区添加至本地仓库
    4. git push -u https://github.com master将本地仓库内容push到远程仓库

Hexo

  1. hexo cl清空缓存
  2. hexo g编译
  3. hexo s启动本地服务
  4. hexo d上传到远程仓库
